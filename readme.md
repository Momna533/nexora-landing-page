# Nexora Saas Landing Page

## 📌 Overview

This project is a responsive web application built with **plain HTML, CSS, and JavaScript**, styled using **Tailwind CSS**.  
It demonstrates modern UI/UX practices such as theme toggling, mobile navigation, pricing tabs, and FAQ accordion functionality.  
The goal is to showcase clean, optimized code and professional frontend development skills.
